Como utilizar o JQuery Animated Modal?

O animatedModal.js é um plugin jQuery para criar um modal de tela cheia com transições CSS3.
Você pode criar suas próprias transições ou usar as transições do animate.css.

Este tutorial, irá lhe ensinar a como utilizar duas dessas transições:

  • Exemplo 1:

    Ao abrir: 'zoom in';

    Ao fechar: 'bounceOut';

    Exemplo 1 - Modal Animado

  • Exemplo 2:

    Ao abrir: 'lightSpeedIn';

    Ao fechar: 'bounceOutDown';

    Como fazer?
Passo 1 - Faça o download do arquivo .zip das extensões do Animated Modal BAIXAR VERSÃO 1.0

Passo 2 - Descompacte o arquivo .zip, retire o arquivo "css" e o arquivo "js" do arquivo animatedModal.js e coloque-os na mesma pasta do seu projeto, chame as extensões abaixo:

            <link rel="stylesheet" href="css/normalize.min.css">
            <link rel="stylesheet" href="css/animate.min.css">
            <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
            <script src="js/jquery.min.js"></script>
            <script src="js/animatedModal.js"></script>
 Passo 3 - No <body> crie um id="corpo" e dois botões para chamar os modais:

            <button type="button" name="button" id="demo01" href="#animatedModal" class="btn btn-secondary"
             onClick="document.all.campoModal.style.display='block'"> Modal </button>

            <button type="button" name="button" id="demo02" href="#modal-02" class="btn"
             style="background-color: darkBlue;color: white;"> Como fazer? </button>
 Passo 4 - Crie a <div> do conteúdo do modal e a <div> dos botões de fechar.

<!--Modal 01-->
            <div id="animatedModal">
              <div  id="btn-close-modal" class="close-animatedModal">
                <!-- BOTÃO DE FECHAR MODAL -->
                <button id="fechar" type="btn button" style="display:none;">Fechar</button>
              </div>

              <!-- Conteúdo do Modal 01-->
              <div class="mt-2 pt-4 pl-4 pr-4" id="campoModal" 
              style="display:none; margin:3%; color: white; height:80%; background-color:darkGray;">
                <p style="text-align:center;">Título do Conteúdo do seu Animated Modal
                <p>Conteúdo do seu Animated Modal!</p>
              </div>
            </div>

            <!--Modal 02-->
            <div id="modal-02">
              <div  id="btn-close-modal" class="close-modal-02">
                <!-- BOTÃO DE FECHAR MODAL -->
                <button id="fechar02" type="btn button" style="display:none;">Fechar</button>
              </div>

              <div id="campoModal02" class="modal-content mt-2 pt-4 pr-4" style="display: none;color: white;">
                <!-- Conteúdo do modal 02 -->
                <p>
                  <div class="col-lg-offset-3 text-left pl-5" style="color:black;">
                    Conteúdo do seu modal aqui
                  </div>
                </p>
              </div>
            </div>
 Passo 5 - No <body> crie dois botões para chamar os modais:

            <script>

            //Configurando o modal 1 por meio do ID demo01
            $("#demo01").animatedModal({
              color:'gray',
              width:'60%',
              height:'60%',
              left: '20%',
              top: '10%',
              animationDuration: '.60s',
            });

            //Configurando o modal 2 por meio do ID demo02
            $("#demo02").animatedModal({
              animatedIn:'lightSpeedIn',
              animatedOut:'bounceOutDown',
              color:'darkBlue',
              width:'60%',
              height:'60%',
              left: '20%',
              top: '10%',
              animationDuration: '.2s',
            });

            </script>
 Passo 6 - Crie um script para fazer com que, ao clicar no botão 2, o campoModal02 apareça e o <body> mude de cor para melhor visualização do seu modal. Por fim, com a tag <style> centralize os botões:

<script type="text/javascript">
            window.onload = function(){
              document.getElementByID("fechar").
               style = "display: block; margin-left:90%; background: white; color: black; margin-top:1%;";
              document.getElementByID("fechar02").
               style = "display: block; margin-left:90%; background: white; color: black; margin-top:1%;";

              var corpo = document.getElementByID('corpo');
              var demo02 = document.getElementByID('demo02');
              var campoModal02 = document.getElementByID('campoModal02');
              var fechar02 = document.getElementByID('fechar02');

              demo02.onclick = function(){
                corpo.style = 'background-color: rgba(50, 65, 100, 90);';
                campoModal02.style.display = 'block';
              }
              fechar02.onclick = function(){
                corpo.style.backgroundColor = 'white';
              }
            }
            </script>

            <style>

            /* Centralizar os botões */
            #demo01{
              margin-left: 45.5%;
              margin-top: 10%;
            }
            #demo02{
              color: white;
              background-color: darkBlue;
              margin-top: 10%;
            }
            </style>
